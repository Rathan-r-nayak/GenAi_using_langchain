**Python Basics and Quiz**

**Notes:**

**Python Basics**

* High-level, general-purpose programming language
* Emphasizes code readability with indentation
* Dynamically type-checked and garbage-collected

**Features**

* Supports multiple paradigms: structured, object-oriented, functional
* Has a typing system (optional)

**History**

* Conceived by Guido van Rossum in the late 1980s as a successor to ABC
* First released in 1991 as Python 0.9.0
* Lead developer until 2018, then "BDFL-emeritus"
* Now led by a Steering Council since 2019

**Popularity**

* Consistently one of the most popular programming languages
* Widely used in machine learning
